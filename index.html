Tic-Tac-Toe Game

The game is played on a 3×3 grid where two players take turns marking spaces with "X" and "O". The objective is to place three marks in a horizontal, vertical, or diagonal row before the opponent. The program checks for winning conditions after every move and declares the winner or a draw accordingly.

Technologies Used: JAVA

Numerology-Calculator

The Numerology Calculator is a Java-based console application that calculates a person's numerology number using their Date of Birth (DOB). The program takes the full date of birth as input (DD/MM/YYYY format), adds all digits together, and reduces the total to a single digit (1–9). Based on the final number, it displays personality traits associated with that number.

Technologies Used: JAVA

Attendence With Facial Recognition

The Attendance System using Facial Recognition is an AI-based application that automatically marks attendance by identifying individuals through their facial features. Instead of manual attendance, the system captures live video through a webcam, detects faces in real time, and matches them with pre-stored images in the database. If a match is found, the system records the attendance along with date and time.

Technologies Used: Python:lib(NumPy,Pandas),Date-Time, OpenCv
